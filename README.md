# DarkSwitch

DarkSwitch is an experimental, educational project for exploring how a Nintendo Switch-style emulator could be structured.

## Status

This repository currently contains the initial project skeleton.

## Planned layout

- `src/` — core implementation
- `include/` — shared headers
- `tests/` — test scaffolding
- `CMakeLists.txt` — build configuration

## Notes

This is a long-term systems project and is not a finished emulator yet.
